Cafetería Morning - Proyecto Final
Descripción del Proyecto

Cafetería Morning es un sitio web de e-commerce interactivo desarrollado como proyecto final del curso de desarrollo web. El proyecto permite a los usuarios explorar productos, agregarlos a un carrito de compras y enviar mensajes de contacto o reseñas.

Este proyecto integra conocimientos de HTML, CSS, JavaScript, y consumo de APIs, y está diseñado para ser responsivo y accesible.

Tecnologías Utilizadas

HTML5: estructura semántica (header, nav, main, footer).

CSS3: estilos responsivos usando Flexbox, Grid y Media Queries.

JavaScript: renderizado dinámico de productos, manipulación del DOM, carrito persistente con localStorage, validación de formularios.

JSON: almacenamiento local de productos.

Formspree: envío de formularios de contacto y reseñas.

Google Fonts: fuentes personalizadas (Viaoda Libre y Love Ya Like A Sister).

Hosting: Netlify / GitHub Pages.

Funcionalidades

Catálogo de Productos

Renderizado dinámico desde un archivo JSON.

Cada producto muestra imagen, nombre, descripción y precio.

Botón para agregar productos al carrito.

Carrito de Compras

Agregar, eliminar y modificar la cantidad de productos.

Cálculo automático del total.

Persistencia con localStorage.

Contador dinámico de productos en el ícono del carrito.

Formulario de Contacto

Validación de nombre, email y mensaje.

Evita caracteres no permitidos y permite acentos.

Envío mediante Formspree.

Formulario de Reseñas

Validación similar al formulario de contacto.

Permite seleccionar calificación por estrellas y dejar una opinión.

Diseño Responsivo y Accesible

Imágenes con alt descriptivo.

Inputs y botones legibles en todos los dispositivos.

Navegación accesible con teclado.

Cómo Ejecutar el Proyecto Localmente

Clonar el repositorio:

git clone https://github.com/tu-usuario/Cafeteria-morning-Final-Version.git


Abrir index.html en el navegador.

Asegurarse de que productos.json esté en la misma carpeta que index.html.

Explorar productos, agregar al carrito y enviar formularios.
